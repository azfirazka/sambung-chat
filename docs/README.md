# Documentation Directory

This directory contains supplementary documentation for the sambung-chat project.

## Structure

```
docs/
├── README.md              # This file - documentation overview
└── diagrams/              # Diagram source files and assets
```

## Main Architecture Documentation

The primary architecture documentation is located in the project root at [../architecture.md](../architecture.md).

## Diagrams Directory

The `diagrams/` directory contains:
- Source files for architecture diagrams
- Exported diagram images
- Diagram-related resources

## Documentation Guidelines

- All diagrams should use formats compatible with GitHub Markdown (preferred: Mermaid.js)
- Keep diagrams in source control alongside code
- Include text descriptions for accessibility
- Use consistent notation across all diagrams
